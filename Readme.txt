By build and run the App, please follow by these steps:

1. creating a virtual environment
2. activate the virtual environment
3. git clone the csource code into the directory
4. go into web folder and run "pip install -r requirements.txt" to install the necessary packages
5. update the database setting in the setting.py file
6. import the attached mock data into database
7. use "./manager runserver" to running backend django server
8. open a new tab and go into the frontend folder
9. run "npm run dev"
10. open the chrome and go to url "localhost:8080" to view the app
